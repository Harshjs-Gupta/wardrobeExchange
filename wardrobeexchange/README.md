# Wardrobe Exchange

A sustainable fashion platform where users can swap, redeem, and give new life to unused clothes. Join our mission to promote sustainable fashion and reduce textile waste while building a vibrant community!

## Features

- **User Authentication**: Secure login and signup with Firebase
- **Item Management**: Add, edit, and delete clothing items
- **Image Upload**: Upload multiple images for each item
- **Browse Items**: Search and filter through available items
- **User Dashboard**: Track your items and swaps
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **UI/UX**: Framer Motion, Lucide React Icons
- **Notifications**: React Toastify

## Getting Started

```bash

# or

# or

# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Setup

Make sure to configure your Firebase project and add the necessary environment variables for authentication, database, and storage.
